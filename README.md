## 🚀 Deploy for Free (3 options)

### Netlify (easiest — 30 seconds)
1. Go to netlify.com/drop
2. Drag the `faruk-v3` folder onto the page
3. Done! Get a free URL like `farukislam.netlify.app`
4. Add custom domain in Settings → Domain Management

### Vercel
```bash
npx vercel
```
Follow prompts — live in 60 seconds.

### GitHub Pages
1. Push folder to a GitHub repo named `farukislam.github.io`
2. Settings → Pages → Deploy from main branch
3. Live at `farukislam.github.io`
